queue[] enqueue(process) dequeue(index) addToGantt(process) checkArrivingProcess()

queue is the array that holds the readyQueue you can iterate through it to find the shortest job or the most priority. but do not update it directly instead use enqueue(process) and dequeue(index) functions

enqueue is a function that adds an item to queue array as well as the DOM takes a process as a parameter

dequeue is a function that removes an item specified by an index from 0 to queue.length returns a process that was removed from the queue

addToGantt will add a process to gantt chart for specified amount of duration and subtract that duration from the remainig time of the specific process and returns it.

checkArrivingProcess() function checks processes that have similar arrival time as the CurrentTime variable : must be called at the end of each function

time function will call the algorithms function every second so as to prevent loop which might cause the browser to run out of memory. time(algorithm) will not be called directly.
